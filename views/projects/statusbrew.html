Brief

Project Description: Intern based work done on a use-case SigCloud, which caters to providing packages of big data services on cluster of nodes/machines/instances launched by a company on cloud network.
Project Status: Front-End Development Complete
Client: Sigmoid, Bengaluru
Guide: Asit Parija
Project Duration: May - Jun'15
Design Responsibilities: Understanding users and their goals, competitive analysis, information architecture, wireframes, icons.
Technical Responsibilities: Basic structure/layout & functionalityof the pages achieved using HTML5, CSS3, Bootstrap, Jquery etc. Advanced usage of Angular.js framework for creating repetitve structures and hierarchy at many places. Use of Angular-$http,$timeout services for continous polling of data from Node.js server.
